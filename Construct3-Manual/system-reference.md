# System object reference
The **System object** is the only object built in to Construct. It provides features to access the runtime engine, and utilities that are useful for most projects. For more information on the System object see [Project Structure](https://www.construct.net/make-games/manuals/construct-3/overview/project-structure) from the Overview section of the manual.
Since the System object provides a lot of features, it is documented in separate sections for its conditions, actions and expressions.

- [System conditions](https://www.construct.net/make-games/manuals/construct-3/system-reference/system-conditions)
- [System actions](https://www.construct.net/make-games/manuals/construct-3/system-reference/system-actions)
- [System expressions](https://www.construct.net/make-games/manuals/construct-3/system-reference/system-expressions)
