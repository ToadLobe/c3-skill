Many plugins share common [conditions](../project-primitives/events/conditions.md), [actions](../project-primitives/events/actions.md) and [expressions](../project-primitives/events/expressions.md), such as for the object's size and position. Rather than repeating the descriptions of these for each plugin, they are listed in this section.

Not all objects use all the common features; some may use only a few of the ones listed here. This is provided as a reference for all the possible features that may be shown to you in Construct rather than describing any particular plugin's features.

- [Common conditions](common-features/common-conditions.md)
- [Common actions](common-features/common-actions.md)
- [Common expressions](common-features/common-expressions.md)

## Other common features

Many objects share other common features. This manual section only covers the common conditions, actions and expressions. For more information on other common features see the manual section on [instances](../project-primitives/objects/instances.md).
