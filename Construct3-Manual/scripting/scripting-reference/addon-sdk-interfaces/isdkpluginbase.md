# ISDKPluginBase addon SDK interface

## On this page
- [ISDKPluginBase APIs](#internalH1Link0)

---
The `ISDKPluginBase` interface is used in the runtime as a base class for plugins created with the addon SDK. It derives from [IPlugin](../object-interfaces/iplugin.md).

## ISDKPluginBase APIs
The `ISDKPluginBase` interface does not currently have any of its own APIs.
