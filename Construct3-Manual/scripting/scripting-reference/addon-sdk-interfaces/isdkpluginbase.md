The `ISDKPluginBase` interface is used in the runtime as a base class for plugins created with the addon SDK. It derives from [IPlugin](https://www.construct.net/make-games/manuals/construct-3/scripting/scripting-reference/object-interfaces/iplugin).

## ISDKPluginBase APIs

The `ISDKPluginBase` interface does not currently have any of its own APIs.
