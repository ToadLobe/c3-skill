# Construct 3 Manual

This directory contains the official Construct 3 manual, converted from [construct.net](https://www.construct.net/en/make-games/manuals/construct-3).

## Overview

Construct 3 is a browser-based game development engine that allows you to create games using a visual event system or JavaScript scripting. This manual provides comprehensive documentation of all features.
