`[Paid plans only]`  Some kinds of [project files](../project-primitives/files.md) added to the [Project Bar](bars/project-bar.md) can be edited directly in Construct using one of the following file editors. See the section on each editor for more information.

- Array files (stored in JSON format) for the [Array object](../plugin-reference/array.md) can be viewed and edited with the [Array editor](file-editors/array-editor.md).
- Dictionary files (stored in JSON format) for the [Dictionary object](../plugin-reference/dictionary.md) can be viewed and edited with the [Dictionary editor](file-editors/dictionary-editor.md).
- Any other text-based file can be viewed and edited with the [Text editor](file-editors/text-editor.md).

### Opening the file editors

To open a file editor, start by adding a new file in the *Files* folder of the [Project Bar](bars/project-bar.md). For more information, see [project files](../project-primitives/files.md).
