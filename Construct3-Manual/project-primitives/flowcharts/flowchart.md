# Flowchart

## On this page
- [Adding flowcharts](#internalH1Link0)
- [Flowchart properties](#internalH1Link1)

---
The flowchart itself does not have a lot of information - in fact it only has a name which is used at runtime by the [Flowchart Controller](../../plugin-reference/flowchart-controller.md) to create instances of flowcharts defined in the editor. Usually it is the nodes inside a flowchart that are of the main interest.

## Adding flowcharts
Right-click the flowcharts folder in the [Project Bar](../../interface/bars/project-bar.md) and select the **Add flowchart** option.
To edit a flowchart, double click on it in the Project Bar or right-click on it and select the **Open** option from the context menu. Doing any of those things will bring up the [Flowchart View](../../interface/flowchart-view.md).

## Flowchart properties
**Name**  
The name of the flowchart. Used at runtime to create an instance of a flowchart that the Flowchart Controller can manipulate.
