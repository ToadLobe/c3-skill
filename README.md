## Construct3-Manual

The main Construct 3 documentation covering:

- **Getting Started** - First steps and tutorials
- **Interface** - Editor UI, dialogs, toolbars
- **Project Primitives** - Layouts, layers, events, objects
- **Plugin Reference** - Sprite, Text, Audio, and 60+ built-in plugins
- **Behavior Reference** - Physics, Platform, Tween, and 30+ behaviors
- **Scripting** - JavaScript/TypeScript API reference
- **Tips & Guides** - Best practices and advanced tutorials

## Construct3-Addon-SDK

Documentation for creating custom addons:

- **Plugins** - Custom object types with actions, conditions, expressions
- **Behaviors** - Reusable logic attachable to objects
- **Effects** - GLSL/WGSL shaders for WebGL/WebGPU
- **Themes** - Custom CSS appearances for the editor
- **Wrapper Extensions** - Native platform integration (Windows/macOS/Linux)
