## Example Team

```json
{
  "teamID": "f827cd64-7325-4cb9-9f18-1a6e74ff0d3e",
  "name": "Red Team"
}
```

## Object Properties

**teamID guid**  
A unique ID for this team.

**name string**  
The name of this team.  Names cannot exceed 64 characters in length.
