## Example Team

```json
{
  "teamID": "dfde08c6-bae6-4702-9561-972c216779a8",
  "name": "Red Team"
}
```

## Object Properties

**teamID** [guid](../../data-types.md#internalH1Link1)  
A unique ID for this team.

**name** `string`  
The name of this team.  Names cannot exceed 64 characters in length.
