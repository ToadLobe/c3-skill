## Example Team

```none
{
  "teamID": "3442b56d-308d-403c-b290-65500600e788",
  "name": "Red Team"
}
```

## Object Properties

**teamID guid**
A unique ID for this team.

**name string**
The name of this team.  Names cannot exceed 64 characters in length.
