## What is the Translation Service?

Various objects in the Construct Game Services can have translatable properties.  For example, [broadcast messages](https://www.construct.net/game-services/manuals/game-services/broadcasts/api-objects/broadcast-message) have a title and text content that can be translated into multiple languages.  When a client makes a request for this content, you can send the [`requestedLanguage`](https://www.construct.net/game-services/manuals/game-services/languages) property with the request.

Your plan has a certain number of monthly translation characters that can be used.  If you have previously translated a translatable property into a specific language, this translation is saved and does not consume any of your translation budget.  If it has not yet been translated and you have enough characters, a new translation will be generated and the result returned to the client.  If you do not have enough translation budget left in your plan, the original source language text will be returned instead.

If you create translatable text and then later update the source text, all translations for this text will be invalidated and new translations will be generated even if only one character has changed.  For this reason you should avoid edits if you want to preserve your translation budget.

## Translation Target Languages

Currently the following language are supported as translation targets (languages that can be translated to):

```none
AR (Arabic)
BG (Bulgarian)
ZH-HANS (Chinese (simplified))
ZH-HANT (Chinese (traditional))
CS (Czech)
DA (Danish)
NL (Dutch)
EN-US (English (American))
EN-GB (English (British))
ET (Estonian)
FI (Finnish)
FR (French)
DE (German)
EL (Greek)
HE (Hebrew)
HU (Hungarian)
ID (Indonesian)
IT (Italian)
JA (Japanese)
KO (Korean)
LV (Latvian)
LT (Lithuanian)
NB (Norwegian Bokmål)
PL (Polish)
PT-PT (Portuguese (all Portuguese variants excluding Brazilian Portuguese))
PT-BR (Portuguese (Brazilian))
RO (Romanian)
RU (Russian)
SK (Slovak)
SL (Slovenian)
ES (Spanish)
ES-419 (Spanish (Latin American))
SV (Swedish)
TH (Thai)
TR (Turkish)
UK (Ukrainian)
VI (Vietnamese)
```

Your plan may have restrictions on the total number of supported translation targets, see your games translation settings for more information on this.

## Translations Source

Translations are generated by [DeepL](https://www.construct.net/out?u=https%3a%2f%2fwww.deepl.com%2f) which generates very high quality and accuracy translations.
