# IZipFileEntry interface

The `IZipFileEntry` interface is an opaque reference to an file entry in [IZipFile](https://www.construct.net/make-games/manuals/addon-sdk/reference/misc-interfaces/izipfile). It has no methods - instead, simply pass it to one of the other methods in IZipFile.
